# Ew

#### 介绍
EW后台管理系统,基于Spring Boot开发简洁的前后端分离权限管理系统

#### 后端技术
- Spring Boot
- Mybatis Plus
- Druid
- Shiro
- EhCache
- Swagger2

#### 使用说明
``` lua
Ew
├── admin -- 后台系统入库
├── common -- 公共代码
├── component -- 可选组件
|    ├── actionLog -- 日志组件
|    ├── Light-Year-Admin-Ui -- 光年后台模板组件
|    ├── shiro -- 权限组件
├── devtools -- 代码生成
├── modules -- 模块
|    ├── system -- 系统模块
